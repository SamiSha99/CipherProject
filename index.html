Simple Encryption System

An Information Security project made by Sami Shakkour, supports 3 different types of Ciphers that can be utilized.

Made using Pure JS through utilizing "FileReader" then reading the files bits as "Uint8Array" before pushing it into one of the algorithms which returns an encrypted result.

Works with any file you would imagine, drop a txt, an image or a video and see for yourself! (Note: If your hardware is weak, don't upload huge files)
